üöÄ Quick Start

  1. Install dependencies: npm install
  2. Set up MySQL database: mysql -u root -p fitness_club_db < schema.sql
  3. Configure .env file with your database credentials
  4. Start the server: npm start
  5. Access API at: http://localhost:3000
